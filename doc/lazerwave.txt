		  Lazerwave color scheme    by Jordan

Lazerwave, a color scheme with a unique touch.

==============================================================================
Color scheme options					lazerwave-options*

						*g:lazerwave_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:lazerwave_transp_bg = 0
<
